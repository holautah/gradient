About this tool

Create animated gradients.

Un generador interactivo y libre de degradados tridimensionales fluidos. Diseñado para crear hermosos fondos orgánicos y con grano para tus proyectos o fondos de pantalla directamente en el navegador. Modifica colores, distorsiones de ruido y dinámicas de movimiento en tiempo real para obtener texturas fluidas perfectas.

—

5 colores de gradiente totalmente personalizables con previsualización WebGL fluida.

—

Estilos con grano (grainy/noise texture) con frecuencia de distorsión dinámica ajustable.

—

Exportación directa en Ultra HD 1080p o 4K para fondos de escritorio o diseños móviles.

—

Guardado rápido mediante URL compartible que guarda cada variable de tu configuración.

—

Presets curados inspiradores y opción de simular una malla tridimensional (Wireframe).
